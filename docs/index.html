SBTCVM Mark 2:

SBTCVM stands for:
Simple Balanced Ternary Computer Virtual Machine

like the name suggests, its a project to create a Balanced Ternary Virtual Machine.  Its primary goals include:
  • Increase understanding and awareness of balanced ternary mathematics and logic.
  • provide Examples for both other Balanced Ternary VM projects, and real attempts at hardware. with the hardware side of things being more logical examples such as balanced ternary color handling and text encoding.
  • Be useful as a tool to study Balanced Ternary computation and Mathematics.

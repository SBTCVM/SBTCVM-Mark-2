SBTCVM Mark 2:

SBTCVM stands for:
Simple Balanced Ternary Computer Virtual Machine

Features:
  • High speed instruction level simulation powers SBTCVM's VM engine.
  • Coded in the programming language: python, and using pygame, SBTCVM is quite portable, and allows fast code testing, easier modification, and simple installation.
  • custom assembly language with keyword-based goto referencing and other features to make development easier.
  • Custom command shell with balanced ternary mathematics operations to expose the power of SBTCVM's underlying integer mathematics routines.
  • various tools and utilities
  • Friendly Graphical main menu and pause menu and detailed HTML documentation to help users get to know SBTCVM and make it easier to use.
  • libbaltcalc, SBTCVM's Powerful Balanced ternary integer mathematics library, provides high-speed conversion to and from decimal and various other operations to power not just SBTCVM, but many utilities and aspects of SBTCVM, such as the command shell's mathematics commands.

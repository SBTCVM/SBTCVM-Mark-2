Overview

SBTCVM is a project to develop a balanced ternary Virtual Machine in python.
You might have a few questions, such as What is balanced ternary?

One goal of SBTCVM is to provide examples and to teach various aspects of balanced ternary mathematics and logic.
SBTCVM uses pygame for graphics, sound, and user input, and is aided by the  libbaltcalc balanced ternary integer mathematics library.


Specifications

  • 6-trit instructions.
  • 9-trit data
  • 19,683 words of executable memory space.
  • 9-trit IO bus.
  • 729 9-trit words of scratch memory.
  • 27x27 6-trit, 729 color RBG display
  • 9x9 2-trit monochrome display
  • 27x36 character TTY
  • buzzer
  • support for up to 6, bank switched sections of executable memory, 19,683 words each.

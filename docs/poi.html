Overview:
This section of SBTCVM's Documentation describes how various possible future features should be implemented, and acts as a general description of implementation Goals for SBTCVM.

Goal Symbol Key: (combinations of some of these may occur)
  • red   "!" = higher priority
  • green "C" = goal for SBTCVM Mark 2.* series.
  • green "L" = longer term goal for beyond Mark 2.* series, but the Mark 2.* series may get some work towards this goal as well.
  • blue  "L" = longer term goal for beyond Mark 2.* series that is highly unlikely to be implemented in the Mark 2.* series for various reasons such as technical and design limitations.
  • blue  "?" = possibility for beyond Mark 2.* series
  • green "?" = possibility for Mark 2.* series
  • violet CB = codebase related
  • violet VG = Vm enGine related

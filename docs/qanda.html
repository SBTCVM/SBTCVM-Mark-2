Q1:
SBTCVM looks complicated... Where should I start?
 A1:
Well... You are looking in the right place. The HTML documentation is the best place to start.  A decent understanding of balanced ternary won't hurt.  Also, you may find the main menu launcher interesting. MK2-MENU.py is a graphical menu with Demos, an introduction, and even a number guessing game that uses SBTCVM's random integer port.
